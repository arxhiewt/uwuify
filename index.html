Commands

Documentation

/uwuify @user

Transforms all messages of the mentioned user into uwu style live.

/webhook #channel

Sets the specified channel for live transformations via webhooks.

/help

Displays the full command list and usage instructions.

Tips:

  • You must have Manage Webhooks permission to use /webhook.
  • Transformations are temporary and don’t store messages permanently.
  • Owner-only commands are protected.

FAQ

Is uwuify free?

Yes! Fully free to invite and use on any server.

Does uwuify store messages?

No messages are stored permanently. Only temporary processing for transformations.

Can I remove uwuify from my server?

Yes, you can remove the bot anytime like any other Discord bot.

Privacy Policy

uwuify does not permanently store message content. User IDs and server IDs are temporarily stored for functionality. No data is sold or shared.
