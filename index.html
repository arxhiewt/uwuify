Setup Information

These permissions are required for uwuify to work properly.

Bot Permissions

  • View Channels
  • Send Messages
  • Read Message History
  • Manage Messages
  • Manage Webhooks

Who Can Use Commands?

  • Administrator
  • Ban Members
  • Kick Members
  • Timeout Members
Ensure your moderation roles include the required permissions. This keeps your server secure and prevents abuse.

Full documentation is still being expanded.

privacy

Privacy Policy

uwuify temporarily processes messages for live uwuification. Messages are deleted immediately after being sent.


The only temporary data stored is which users are currently being uwufied. This data is automatically removed once the command ends or is manually stopped.


No permanent storage of messages, user IDs or server data occurs.
